Stratum local release

Dev server only. Bumps package.json, writes a commit with subject = version and body [Summary] / [Changes] (same as release-commit.sh). Uses your local Git identity and repo — nothing is pushed from here.

Your dev server was started with STRATUM_UPDATE_TOOL_TOKEN in .env.local (repo root). Paste the same secret here so preview/commit requests are allowed. This blocks random websites from POSTing to your machine while this tab is open — it is not your Git password and nothing is sent off-device.
